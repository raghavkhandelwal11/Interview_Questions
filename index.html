Difference in div and span

Div: is block level element and it is bound to have a whole width of a screen It is used for grouping the elements in HTML documents, means div is used to wrap section of documents span: is a inline element, means it only takes the space required for the content in it. span should be used to wrap small portion of text and images etc.
